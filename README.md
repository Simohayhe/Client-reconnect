Client-reconnect は、サーバーから切断された際に、タイトル画面に戻ることなく自動的に再接続を試みるための、軽量なFabric/Quilt用クライアントサイドMODです。

🌟 特徴
自動再接続: サーバーの再起動やネットワークの一時的な不安定さで切断された際、ボタンを押さずに再接続を開始します。

待機時間の表示: 次の再接続試行までのカウントダウンを表示し、状況を把握しやすくします。

キャンセル機能: もし再接続を止めてタイトル画面に戻りたい場合は、いつでもキャンセルボタンで戻ることが可能です。

軽量設計: ゲームのパフォーマンスに影響を与えず、バックグラウンドで静かに動作します。

✨ 導入のメリット
放置（AFK）に最適: トラップタワーでの放置中などに回線が切れても、自動で復帰できる可能性が高まります。

サーバー再起動時の手間削減: 定期メンテナンス等でサーバーが再起動した際、手動でログインし直す手間が省けます。

🔧 導入方法
ビルドされた .jar ファイルを、自身のPCの .minecraft/mods フォルダに配置します。

Minecraft（Fabric/Quilt環境）を起動します。

マルチプレイサーバーに接続中、切断が発生すると自動的に再接続画面が表示されます。

📋 動作要件
Loader: Fabric / Quilt

Environment: Client-side (クライアント側のみに導入してください)
